WordPress em Alta Disponibilidade na AWS

Implementação de uma infraestrutura de nuvem robusta e segura para hospedar WordPress, totalmente automatizada com Infraestrutura como Código.

AWS Segurança Docker IaC
O Desafio

Hospedar um site WordPress garantindo operação contínua, performance e segurança, mesmo sob picos de tráfego ou falhas de hardware.

CI/CD com GitHub Actions, Docker e ArgoCD

Pipeline GitOps para automatizar o ciclo de vida de uma aplicação, do `git push` à implantação em Kubernetes, com foco em agilidade e confiabilidade.

CI/CD Docker Kubernetes
O Desafio

Automatizar todo o ciclo de vida de uma aplicação, desde a alteração no código até sua implantação em produção, utilizando uma abordagem GitOps onde o Git é a única fonte da verdade.

Fluxo Automatizado

Este projeto demonstra como empresas modernas mantêm agilidade e confiabilidade em seus deploys, integrando ferramentas líderes de mercado para criar um pipeline seguro e observável.

GitHub Repo (Código)

ArgoCD (CD)

  • CI (Integração Contínua) com GitHub Actions: A cada `git push`, o pipeline automatiza o build da aplicação em um contêiner Docker, envia a imagem para o Docker Hub e atualiza o repositório de manifestos com a nova tag.
  • CD (Entrega Contínua) com ArgoCD: Utilizando a metodologia GitOps, o ArgoCD monitora o repositório de manifestos. Ao detectar uma alteração, ele atualiza automaticamente a aplicação no cluster Kubernetes.
  • Kubernetes: Orquestra a execução dos contêineres, garantindo escalabilidade, resiliência e gerenciamento eficiente dos recursos da aplicação.
